# Code of Conduct

This project expects professional, respectful collaboration.

## Expected Behavior

- Be direct, factual, and considerate.
- Assume good intent, but accept correction quickly.
- Keep feedback focused on code, docs, design, and user impact.
- Respect maintainers' time by providing clear reproduction steps and focused pull requests.

## Unacceptable Behavior

- Harassment, threats, abuse, or discriminatory language.
- Posting private information without permission.
- Repeatedly derailing issues or pull requests.
- Publishing credentials, tokens, customer data, or exploit details outside the security process.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; restrict participation; or ban accounts when needed to protect the project.

Report conduct concerns to `security@polymetrics.ai`.
