# ChatGPT Image Design Prompt

Create a high-fidelity product UI design concept for github-projects-discussions.

Product context:
- Domain: product software
- Primary users: target users
- Main job: primary user flow
- First screen or flow: primary user flow
- Platform: web

Output requirements:
- Show the actual application screen, not a marketing landing page.
- Include realistic data and states.
- Include primary navigation, key controls, empty/error/loading states where relevant.
- Avoid generic stock imagery, decorative blobs, and unreadable tiny text.
- Make the design practical for implementation in a real frontend.
- Design for WCAG 2.2 AA contrast and keyboard-friendly UI.
