# Governance

Polymetrics CLI is maintained by Polymetrics AI.

## Maintainers

Maintainers are responsible for project direction, release decisions, issue triage, security response, and final review on pull requests.

Initial maintainer:

- Karthik Sivadas

## Decision Process

Routine changes are reviewed through pull requests. Larger changes should start with an issue or discussion covering the user problem, proposed design, compatibility impact, and verification plan.

Maintainers may decline changes that expand scope, weaken local-first safety guarantees, add unbounded dependencies, or conflict with the licensing and commercial direction of the project.

## Releases

Releases are cut from `main` after CI passes. Public release artifacts should be reproducible from the tagged source tree.
